## REST API with Spring Boot and MongoDB
Project to study REST APIs with Spring Boot and MongoDB

This project was created to study:

* Java Spring Boot
* Rest API
* Domain Model
* Structure Logical Layers (resource, service, repository)
* MongoDB/NoSQL
* CRUD
* Maven
* Handling Exceptions
